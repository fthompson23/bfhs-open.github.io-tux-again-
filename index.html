BFHS-Open About TUX KART

A center of gravity for open work happening at or associated with Benjamin Franklin High School.

BHFS-Open is an initiative of the BFHS Computer Science Club in effort to create, organize, and support projects being done transparently.

Get involved

  • Attend weekly meetings in Room 213
    • Currently Mondays after school
  • Join the CSC Discord Server
  • Join our IRC channel at ##bfhs-open on libera.chat

How to participate

  • Contribute to our various projects on Github
  • Create your own knowledge base and share it with us

Projects

  • Official BFHS-Open website
  • Collective knowledge base
  • SuperTuxKart game server
  • Learning Git to keep track of changes
  • Learning Python to make games, utilities, and more
